# Spam Email Detection

This project builds a simple machine learning model to classify emails as **spam** or **not spam**.

## Idea
Email text is converted into numbers using **CountVectorizer**, then a **Logistic Regression** model is trained to detect spam.

## Steps
1. Load the dataset
2. Split data into training and testing
3. Convert text to numbers
4. Train the model
5. Evaluate the results

## Goal
Understand the basics of **text classification using machine learning**.

dataset: https://www.kaggle.com/datasets/jackksoncsie/spam-email-dataset
